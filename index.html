Jutro kolejna Warsjawa

11 października 2013

Już jutro będzie miała miejsce kolejna edycja Warsjawy. Niestety w momencie startu rejestracji byłem jeszcze w trakcie urlopu.... Piszę niestety ponieważ wiem, że wolne miejsca znikają szybciej niż ostatnia wersja ajfona ze sklepów. Nie myliłem się, po powrocie do kraju zostało bardzo mało warsztatów do wyboru. W tym roku cały dzień został podzielony na 3 sloty czasowe. Zdecydowałem się na warsztaty Gradle zajmujące 2 pierwsze sloty, które będą prowadzone przez Michała Opałę i Arkadiusza Koniora. Ostatni slot, zostawiłem na networking.

Włączyłem komputer, założyłem słuchawki, odpaliłem na dwa sety i zabrałem się do przygotowania środowiska. Zgodnie z wiadomością wysłaną przez Michała potrzebne będą Java JDK v.1.7, Gradle 1.8, Groovy, podręcznik Gradle, zawartość repozytorium z Githuba (a co za tym idzie i Git) oraz wygodny edytor. Sprawdziłem czy mam wszystkie niezbędne narzędzia. Okazało się, że aktualizacji wymagały gradle i groovy. Pobieżne przejrzenie plików .gitignore pozwoliło mi wywnioskować, że najlepiej jeśli tym "wygodnym edytorem" będzie IntelliJ IDEA. Od jakiegoś czasu bardzo przypadł mi do gustu Sublime Text. Uznałem jednak, że warsztaty to dobry pretekst, żeby wreszcie spróbować pobawić się IDE, które zachwala tak wielu znajomych deweloperów. Całość przygotowania, wraz z instalacją intellij i dociągnięciem zależności zdefiniowanych w dependencies.gradle zajęła mi ok. 30 min i poszła bezproblemowo. Całkiem szybko, byłem przekonany, że zejdzie się dłużej...

Do zobaczenia jutro rano.

How to turn off data transmission via GSM/3G in Windows Phone 8?

18 września 2013

1) Turn off data transmission:

Settings > Mobile Network > Data Transmission > Off

2) E-mail & other accounts

Settings > E-mail & Accounts > Account (Google/Facebook/Twitter/etc) >
> Download new content > Manually

3) Navigation

Here Drive+ > Online connection > Offline

4) Facebook

Settings > Messages > Use Facebook chat > Off

5) Lock screen

Settings > Locking screen > Notifications > Detailed > None
Settings > Locking screen > Notifications > Short > None

6) Background tasks

Settings > Background Tasks > [App] > Blocked

7) Photos & Camera

Settings > Applications > Photos & Camera >
> Add localizations to photos > Off

Additionally You can turn off Wi-Fi

Settings > Wi-Fi > Wi-Fi Networks > Off

How to set up basic git config?

18 lutego 2013

basic & obligatory

set user/commiter name

git config –global user.name “Kapitan Bomba”

set user/commiter email

git config –global user.email “kapitan.bomba@youtu.be”

hint: type git config without “–global” option if you want
to have settings specific only for current repository
(it’ll write to project/.git/config file).

setting up nice colors

git config –global color.diff auto
git config –global color.status auto
git config –global color.branch auto

setting up default merge tool

$ git config –global merge.tool vimdiff

or: kdiff3, tkdiff, meld, xxdiff, emerge, vimdiff, gvimdiff, ecmerge, opendiff

you can also setup a custom tool (fixme – how to do this???)
